Some useful tools:
===

project.sh              test project build script
item.sh                 test item build script
suite.sh                deprecated. Use project.sh instead
case.sh                 depreacted. Use item.sh instead
ignore.sh               mark an item to be ignored by the run.sh
list.sh                 list all current item and its document (also list ignored item)
update.sh               update necessary files for project/item after a svn update
jget.sh                 get configuration of Juniper router
